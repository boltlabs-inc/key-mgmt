Authenticated
Passed to channel type as the AUTH generic parameter. It is used for channels that handle authenticated operations. This type ensures that messages moving across a channel are encrypted.
Channel
A two-way channel between a client and server used to communicate with Message objects. Channel uses tonic types to receive messages. It can only be used with streaming messages generated by tonic.
Unauthenticated
Passed to channel type as the AUTH generic parameter. It is used for channels that handle unauthenticated operations. This type does nothing to modify messages passed across a channel.
